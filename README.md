# crud_akademik
Projek Latihan CRUD dengan Framework Laravel 
by: arispras@gmail.com

Latihan membuat aplikasi sederhana mulai dari 
1. Login dengan laravel Auth + Bootstrap tanpa Laravel Mix.  
2. Crud data jurusan.
3. Crud data mahasiswa.

   Note:
   Totorial langkah2 membuat aplikasi ini dapat dilihat pada file: Tutorial.docx yang ada pada project ini.
   Database di dump pada file akademiksaintek.sql
